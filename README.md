# ACC102 Track 2: Retail Sales Analysis

## 1. Project Purpose
This project analyses retail sales data to identify key business trends, regional performance, and product category popularity. The intended users are retail managers who need clear, data-driven insights to support inventory and marketing decisions.

## 2. Dataset
Simulated daily retail sales data for 2023, including:
- Date
- Daily sales amount
- Customer count
- Region (North, South, East, West)
- Product category (Electronics, Apparel, Beauty, Home)

## 3. Tools & Libraries
- Python
- Pandas (for data processing)
- Matplotlib (for data visualisation)
- GitHub (for project hosting)

## 4. Analysis Methods
1.  Data cleaning and summary statistics
2.  Monthly sales trend analysis
3.  Regional sales performance comparison
4.  Product category sales analysis
5.  Visualisation of key findings

## 5. Key Findings
- The **East region** achieves the highest total sales, making it the strongest performing market.
- Sales show a clear upward trend, with a peak in **December**, likely driven by holiday shopping demand.
- **Electronics** is the top-performing product category, generating the highest revenue.

## 6. How to Run
1.  Install the required dependencies: `pip install -r requirements.txt`
2.  Open `ACC102 Python analysis.ipynb` in Jupyter Notebook
3.  Run all cells to view the analysis and visualisations

## 7. Limitations & Improvements
- The dataset is simulated for educational purposes; real transaction data would provide more accurate insights.
- Future improvements could include adding time-series forecasting and interactive dashboards.
