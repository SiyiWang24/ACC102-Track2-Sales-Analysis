# ACC102 Track 2: Retail Sales Analysis

## 1. Project Purpose
This project analyses retail sales data to identify business trends, regional performance, and product popularity. The intended users are retail managers who need data-driven insights.

## 2. Dataset
Simulated daily sales data including date, sales amount, customer count, region, and product category.

## 3. Tools & Libraries
- Python
- Pandas
- Matplotlib
- GitHub

## 4. Analysis Methods
- Data cleaning and summary statistics
- Monthly sales trend analysis
- Regional sales comparison
- Product category performance analysis
- Data visualisation

## 5. Key Findings
- The East region achieves the highest total sales.
- Sales peak in December due to seasonal holiday demand.
- Electronics is the top-performing product category.

## 6. How to Run
Open analysis.ipynb and run all cells to view results and charts.

## 7. Limitations
The data is simulated for educational purposes. Real business data would provide more accurate insights.
